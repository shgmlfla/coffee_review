## coffee_review 사이트
카페 리뷰 사이트
## 개요
인스타그램 같은 SNS가 발달하면서 카페 공유가 활발해졌고 방문자가 사용가능한 정보의 범위도 증가했다. 카페 리뷰 사이트를 사용자가 원하는 
필터에 맞게 조회하고 커뮤니티 기능을 통해 카페 리뷰에 대해 공유할 수 있다.

## 개발환경
```
* RDBMS : ORACLE
* 
```

## 사용 기술
Spring, Java, Oracle, Git, Oracle cloud
